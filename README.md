# Watsons Teşhir Uygulamaları

Watsons mağazalarındaki görsel düzenlemelerin ve insert yerleşimlerinin takibi için geliştirilmiş web uygulaması.

## 🚀 Özellikler

- 🏪 Mağaza personeli için kolay kullanımlı arayüz
- 👥 Bölge müdürleri için değerlendirme sistemi
- 📸 Görsel yükleme ve karşılaştırma
- 📊 Detaylı raporlama ve analiz
- 🔄 Gerçek zamanlı güncelleme
- 🔒 Güvenli veri yönetimi

## 🛠️ Teknolojiler

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase
- React Router
- GitHub Actions

## 🚦 Başlangıç

1. Repoyu klonlayın:
```bash
git clone https://github.com/kullaniciadin/watsons-teshir-uygulamalari.git
cd watsons-teshir-uygulamalari
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. Çevre değişkenlerini ayarlayın:
```bash
cp .env.example .env
# .env dosyasını düzenleyin
```

4. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```
