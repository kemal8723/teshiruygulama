# Watsons Teşhir Uygulamaları

Bu uygulama, Watsons mağazalarındaki görsel düzenlemelerin ve insert yerleşimlerinin takibi ve onayı için geliştirilmiş bir web uygulamasıdır.

## Özellikler

- Mağaza personeli ve bölge müdürleri için ayrı arayüzler
- Görsel yükleme ve karşılaştırma
- Gerçek zamanlı değerlendirme sistemi
- Detaylı raporlama ve takip

## Teknolojiler

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase

## Kurulum

1. Repoyu klonlayın:
   ```bash
   git clone https://github.com/yourusername/watsons-teshir-uygulamalari.git
   ```

2. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

3. `.env.example` dosyasını `.env` olarak kopyalayın ve Supabase bilgilerinizi ekleyin:
   ```bash
   cp .env.example .env
   ```

4. Uygulamayı başlatın:
   ```bash
   npm run dev
   ```
